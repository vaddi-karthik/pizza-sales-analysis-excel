# Pizza Sales Analysis Dashboard | Microsoft Excel

Excel-based sales analytics dashboard using Pivot Tables, XLOOKUP, KPI cards, and data visualization.

## Project Overview
Developed an interactive Excel dashboard to analyze pizza sales performance, customer ordering patterns, and revenue trends using business reporting and data visualization techniques.

## Objectives
- Analyze customer ordering behavior and sales trends
- Track KPI metrics and revenue performance
- Identify top-performing pizza categories and products
- Discover peak sales hours and business insights

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- SUMIFS
- KPI Reporting
- Slicers
- Conditional Formatting
- Data Validation

## Dataset Information
- Analyzed 21,350 customer orders
- Evaluated $817K+ annual revenue
- Integrated data from 4 relational datasets

## Key Insights
- Peak sales hours observed between 12–1 PM and 5–7 PM
- Certain pizza categories contributed significantly to revenue
- Monthly trend analysis identified sales performance patterns
- Dashboard enabled interactive business performance monitoring

## Dashboard Features
- KPI Cards
- Dynamic Charts
- Interactive Slicers
- Revenue Analysis
- Sales Trend Visualization
- Product Performance Analysis

## Project Outcome
This project demonstrates practical skills in Excel analytics, dashboard development, KPI reporting, data cleaning, and business reporting.
